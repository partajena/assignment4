Differences between HTML and HTML5

HTML HTML5
HTML is based on SGML (Standard Generalized Markup Language). HTML5 is not based on SGML. It is a standalone markup language.
HTML does not support new semantic elements like `
`, `
`, `
`, etc.
 HTML5 introduces new semantic elements that provide better structure to web pages.
HTML does not have built-in support for multimedia elements like audio and video. HTML5 includes native support for audio and video, allowing them to be embedded directly in the page without the need for plugins.
